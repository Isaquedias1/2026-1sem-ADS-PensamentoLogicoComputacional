7.1 Criando e Escrevendo em Arquivos de Texto

7.1.1 Criando uma variável “arquivo”: Função interna open()

# 1. Abre o arquivo (se não existir, ele cria)
variavel_arquivo = open("exemplo_classico.txt", "w", encoding="utf-8")

A função open(“nome_do_arquivo”, modo_de_operação, mapa_de_caracteres ) cria um arquivo no disco e associa uma variável a esse arquivo do disco.

7.1.2 Os 3 parâmetros da função open()

A função tem 3 parêmetros:

  • nome_do_arquivo: nome do arquivo no disco;
  • modo_de_operação: se o conteúdo deve adicionar ou sobreescrever o conteúdo anterior;
  • mapa_de_caracteres: se os caracteres são “americanos” (ascii), “globais” (utf-8) ou globais-windows (utf-16)
Operador da função “write” Acrônimo em Inglês Como a função trata o arquivo
‘w’ Write Abre para escrita. Sobrescreve o arquivo se ele já existir.
‘a’ Append Abre para escrita. Mantém o conteúdo e adiciona ao final.
‘x’ eXclusive Cria um novo arquivo. Se o arquivo já existir, o Python retorna um erro.
‘w+’ Update Abre para escrita e leitura simultaneamente (sobrescrevendo).

7.1.3 Escrevendo no arquivo com open(): Função interna write

Toda variável do tipo “arquivo” tem uma função interna chamada “write()”. Essa função insere texto (strings) dentro de um arquivo de texto.

variavel_arquivo.write("Escrita no modo clássico.\n")

7.1.4 Escrevendo no arquivo com open(): Fechando e escrevendo com a função interna Close()

variavel_arquivo.close()

7.1.5 Um programa completo - modo clássico

Exemplo de um programa completo para criar um arquivo e inserir conteúdo dentro dele.


# 1. Abre o arquivo (se não existir, ele cria)
variavel_arquivo = open("exemplo_classico.txt", "w", encoding="utf-8")

# 2. Realiza as operações de escrita
variavel_arquivo.write("Escrita no modo clássico.\n")
variavel_arquivo.write("É necessário fechar o arquivo manualmente.")
print("Dados enviados para a variável do arquivo.")

# 3. Fecha o arquivo obrigatoriamente
variavel_arquivo.close()
print("Arquivo criado e dados salvos com sucesso!")

7.1.6 Simplificando a escrita de arquivo com recursos PYTHON: comando WITH

O comando with elimina a necessidade da criação explícita da variável de arquivo.

Desta forma, ao executar o escopo do comando, o arquivo é aberto, pode-se escrever nele com a função write(), e , ao terminar a escrita, o arquivo se fecha automáticamente.

with open("exemplo.txt", "w", encoding="utf-8") as arquivo:
    arquivo.write("Olá! Este é um exemplo de escrita.\n")
    arquivo.write("Segunda linha do arquivo.")

print("Arquivo gravado com sucesso!")

7.2 Criando e Escrevendo em Arquivos de Texto FORMATADOS

7.2.1 o PADRÃO CSV (VALORES SEPARADOS POR VÍRGULA)

Um arquivo CSV (Comma-Separated Values, ou valores separados por vírgula) é um formato de arquivo texto usado para armazenar dados em forma de tabela, organizados em linhas e colunas.

Arquivos CSV podem ser abertos em programas de planilha eletrônica como EXCEL.

Exemplo 3 - Crie uma planilha de excel usando criando um arquivo texto no formato CSV. A primeira linha deve ser o nome das colunas. A segunda linha deve ser o registro do Cliente. Use os dados da cliente “Ana”, de 25 anos e que mora em São Paulo.

usuarios = [
    ["Ana", 25],
    ["Carlos", 30]
]

with open("usuarios.txt", "w") as arq:
    for nome, idade in usuarios:
        arq.write(f"{nome},{idade}\n")

7.2.1.1 Saída exemplo 3

Ana,25
Carlos,30

Exemplo 4 - Crie uma planilha de excel usando criando um arquivo texto no formato CSV. Crie uma variável lista contendo 2 dicionários. Salve esse conetúdo das variáveis dentro de um arquivo. DICA: Use as facilidades do Python.

import csv

variavel_dicionario = [
    {
        "nome": "Ana",
        "idade": 25,
        "cidade": "São Paulo"
    },
    {
        "nome": "Pedro",
        "idade": 11,
        "cidade": "São Paulo"
    }
]

with open("exemplo2.csv", "w", newline="", encoding="utf-8") as variavel_arquivo:
    formatador_csv = csv.writer(variavel_arquivo)
    
    # escrevendo a primeira linha: cabeçalho (colunas)
    formatador_csv.writerow(variavel_dicionario[0].keys())
    
    # escrevendo a segunda linha: valores (linha)
    formatador_csv.writerow(variavel_dicionario[0].values())
    
    # escrevendo a segunda linha: valores (linha)
    formatador_csv.writerow(variavel_dicionario[1].values())

7.2.1.2 Saída exemplo 4

nome,idade,cidade
Ana,25,São Paulo
Pedro,11,São Paulo

7.3 Comparação entre formatos dados tipo texto: Texto puro, CSV e JSON

Característica TXT (puro) CSV JSON
Estrutura Livre Tabular Hierárquica
Organização Nenhuma (definida pelo programador) Linhas e colunas Objetos e listas
Padronização ❌ Não possui ✔ Simples ✔ Forte (padrão de mercado)
Facilidade de leitura Média Alta Alta
Facilidade de escrita ✔ Muito fácil ✔ Fácil ✔ Fácil (com biblioteca)
Preserva tipos (int, string) ❌ Não ❌ Não ✔ Sim
Uso em sistemas Baixo Médio Alto
Uso em APIs ❌ Não ❌ Raro ✔ Padrão
Flexibilidade ✔ Muito alta ❌ Limitada ✔ Alta

7.3.1 Onde usar cada formato de arquivo para armazenar dados:

  • CSV ↔︎ planilhas de excel, tabelas de banco de dados
  • JSON ↔︎ documentos (NoSQL, APIs)
  • TXT ↔︎ arquivos de logs e arquivos simples

Exemplo 9 - Use as facilidades do Python. Leia a planilha CSV que você crou no exemplo 4

import csv

variavel_dicionario_lido = []

with open("exemplo4.csv", "r", encoding="utf-8") as v_arquivo:
    leitor_csv = csv.reader(v_arquivo)    
    cabecalho = next(leitor_csv)  # lê a primeira linha (colunas)    
    for linha in leitor_csv:
        registro = {}        
        for i in range(len(cabecalho)):
            valor = linha[i]            
            # conversão de tipo
            if valor.isdigit():
                valor = int(valor)            
            registro[cabecalho[i]] = valor
        variavel_dicionario_lido.append(registro)

print(variavel_dicionario_lido)
